To perform a basic connection to a FTP:

```console
zero@pio$ ftp <ip>
```

Inside, basic commands can be use:

| **Command**   | **Description**    |
|--------------- | --------------- |
| `ls`   | List files and folders   |
| `cd`   | Change directory   |
| `get <file>`   | Download file in our machine   |
| `exit`   | Exit the FTP   |
