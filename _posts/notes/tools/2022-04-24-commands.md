# cURL
Curl can be use to send a basic http request to any URL.
```console
zero@pio$ curl zeropio.github.io
```

### Flags
- **-O**: download page.
- **-o**: specify file name.
- **-s**: silent request.
- **-k**: skip the certificate check for https.
- **-v**: verbose.
- **-vvv**: extra verbose.
