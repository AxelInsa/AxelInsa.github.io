# [](#header-4)Connection

# [](#header-4)Flag

Now we are in, we just need to list all the databases and search for the flag.
{% highlight plain %}
MariaDB [(none)]> show databases;
+--------------------+
| Database           |
+--------------------+
| htb                |
| information_schema |
| mysql              |
| performance_schema |
+--------------------+
MariaDB [(none)]> use htb;

MariaDB [htb]> show tables;
+---------------+
| Tables_in_htb |
+---------------+
| config        |
| users         |
+---------------+

MariaDB [htb]> select * from config;
{% endhighlight %}
